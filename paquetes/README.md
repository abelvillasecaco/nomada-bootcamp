# paquete-abel

Pequeñas utilidades JavaScript para practicar la publicación de paquetes npm.

## Funciones disponibles

- `capitalize(text)`: convierte la primera letra en mayúscula.
- `truncate(text, length)`: corta el texto a un número de caracteres.
- `isEmpty(value)`: determina si un valor está vacío.

## Instalación
```bash
npm install paquete-abel
